# RAG App to Mind Map Converter

This project converts the structure and logic of a Retrieval-Augmented Generation (RAG) application into a visual **mind map** using `matplotlib`. It uses Google's **Gemini API** to understand the components and their relationships.

## Features

- 🔍 Parses and analyzes the components of a RAG app (Retriever, Index, Embeddings, LLM, etc.)
- 🧠 Uses Gemini to extract semantic meaning and suggest structure
- 🗺️ Visualizes the app architecture as a mind map using `matplotlib`

## Requirements

- Python 3.8+
- `matplotlib`
- `google.generativeai` (Gemini API SDK)
- `networkx` (optional, for graph building)
